# jCode Syntax Highlighter
Plugin is based on the modular library of [Prism JS](https://prismjs.com).  
Languages: `C`, `C++`, `C#`, `CSS`, `HTML/XML`, `Java`, `Javascript`, `LESS`, `MathML`, `PHP`, `Python`, `Ruby`, `SQL`, `SVG`.
