# 14.11.2020 | Версия 1.2.0
- плагин обновлен до версии Prism JS 1.22.0, https://github.com/prismjs/prism/blob/master/CHANGELOG.md#1220-2020-10-10,
- добавлен скрипт для проверки минимальных системных требований (Joomla 3.9, PHP 7.2),
- обновлены названия классов для Joomla 3.9 / 4.0 (API),
- исправлена ошибка вызова неопределенного метода для Joomla 4.0.

# 03.02.2020 | Версия 1.0.1
- плагин обновлен до версии Prism JS 1.19.0, https://github.com/prismjs/prism/blob/master/CHANGELOG.md#1190-2020-01-13,
- добавлен язык: LESS,
- добавлены локализации: английская (en-GB), украинская (uk-UA).

# 27.12.2019 | Версия 1.0.0
- плагин создан на основе модульной библиотеки Prism JS 1.18.0, https://github.com/prismjs/prism/blob/master/CHANGELOG.md#1180-2020-01-04,
- добавлены скрипты: line numbers, autolinker, previewers, copy to clipboard button,
- добавлены языки: C, C++, C#, CSS, HTML/XML, Java, Javascript, MathML, PHP, Python, Ruby, SQL, SVG,
- добавлены шаблоны: default, dark, okaidia, tomorrow night,
- добавлен сервер для автоматического обновления плагина в панели управления.
