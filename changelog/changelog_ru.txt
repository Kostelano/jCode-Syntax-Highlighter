# 03.02.2020 Версия 1.0.1
- плагин обновлен до версии Prism JS 1.19.0,
- добавлен язык: LESS,
- добавлены локализации: английская (en-GB), украинская (uk-UA).

# 27.12.2019 Версия 1.0.0
- плагин создан на основе модульной библиотеки Prism JS 1.18.0, https://prismjs.com,
- добавлены скрипты: line numbers, autolinker, previewers, copy to clipboard button,
- добавлены языки: C, C++, C#, CSS, HTML/XML, Java, Javascript, MathML, PHP, Python, Ruby, SQL, SVG,
- добавлены шаблоны: default, dark, okaidia, tomorrow night,
- добавлен сервер для автоматического обновления плагина в панели управления.
